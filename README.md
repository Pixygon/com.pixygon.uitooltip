# Pixygon — UI Tooltip

A lightweight hover-tooltip system for UI.

## Key types

| Type | What it is |
|---|---|
| **`TooltipSystem`** | Singleton that shows/hides the tooltip. |
| **`Tooltip`** | The tooltip view. |
| **`TooltipTrigger`** | Drop on a UI element to show a tooltip on hover/focus. |

## Dependencies

None.

## Status

`0.5.0`. UI utility; used by `pagedcontent`.
